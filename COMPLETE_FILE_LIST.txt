📱 FRONTEND CORE:
────────────────────────────────────────────────────────────────────────────
App.js
- Main entry point with navigation setup

package.json

app.json
- Expo configuration

- Required permissions (Location, SMS, Phone)

🏠 SCREENS:
────────────────────────────────────────────────────────────────────────────
HomeScreen.js
- Displays 14 service categories
- Handles location detection
- Opens nearby shops modal
- Navigates to Woman Safety & Blood Donation

WomanSafetyScreen.js (800 lines)
- SOS Emergency Alert System
- Emergency contact management (add/delete/call)
- Find nearby police stations & hospitals
- Quick dial: 100 (Police), 108 (Ambulance), 1091 (Women)
- Location sharing via SMS
- Safety settings & tips

BloodDonationScreen.js (900 lines)
- Emergency blood request system
- Blood donor registration (A+, B+, O-, AB-, etc.)
- Search donors by blood group & radius
- Find nearby blood banks
- Compatible blood group finder
- Donor availability toggle
- Emergency numbers: 108, 102

NearbySettingsScreen.js (600 lines)
- Customize search distance (1km - 10km)
- Set max travel time (5min - 20min)
- Minimum rating filter (0 - 4.5★)
- Display preferences (show/hide distance, ratings, photos)
- API settings (Google Places, OSM fallback, timeout)
- Location accuracy toggle

- Provides auth state to all screens

ThemeContext.js

- Toggle theme function

- Provides theme to all screens

================================================================================
                        TROUBLESHOOTING
================================================================================

❌ "Cannot find module"
   → Run: npm install

❌ "Invariant Violation"
   → Check all imports match file names exactly

❌ "Location permission denied"
   → Enable location in device settings

❌ "Metro bundler error"
   → Run: npx expo start -c --reset-cache

❌ Context errors
   → Make sure AuthContext.js and ThemeContext.js are in src/context/

❌ Navigation errors
   → Check all screen names in App.js match navigation.navigate() calls
